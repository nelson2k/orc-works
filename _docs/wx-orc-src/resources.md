# Resources

Non-code assets in `wx-ocr-src/`.

## Icons

`wx-ocr-src/icons/` — SVG icons. Loaded at runtime by
`wxBitmapBundle::FromSVGFile`. CMake copies the whole folder next to the
exe as a post-build step (see [build.md](build.md)).

| File                    | Used for                          |
|-------------------------|-----------------------------------|
| `OCR_toolbar_icon.svg`  | app/window icon (blue, see below) |
| `folder-open.svg`       | Open PDF button                   |
| `file-text.svg`         | Extract Page                      |
| `files.svg`             | Extract PDF                       |
| `chevron-left.svg`      | Prev page                         |
| `chevron-right.svg`     | Next page                         |

Resolution: `loadIcon()` in [main.cpp](main.md) builds the path as
`<exe-dir>/icons/<name>.svg` via `wxStandardPaths::GetExecutablePath()`.

## App icon (`OCR_toolbar_icon.svg`)

Recolored copy of `_docs/OCR_toolbar_icon.svg` with `stroke="#2196F3"`
(Material Blue 500). Used two ways:

- At runtime, [main.cpp](main.md) rasterizes it via `wxBitmapBundle` at
  16/24/32/48/64/128/256 px, packages those into a `wxIconBundle`, and
  passes it to `SetIcons()` — gives a crisp icon at any DPI.
- At build time, the same geometry is baked into
  [src/icon.ico](../../wx-ocr-src/src/icon.ico), which is embedded in
  the exe via `orcgui.rc` so Windows Explorer, the taskbar shortcut,
  and Alt-Tab fall back to a sensible icon if the SVG is missing.

The `.ico` is generated by a one-off Python/Pillow script that draws
the same lines/strokes as the SVG; it isn't part of the build.

## Win32 resource

[src/orcgui.rc](../../wx-ocr-src/src/orcgui.rc) declares one icon
resource and pulls in `wx/msw/wx.rc`, which provides the manifest with
`wxUSE_RC_MANIFEST 1` and `wxUSE_DPI_AWARE_MANIFEST 2`. That's what
makes the app per-monitor DPI aware on Windows.

[src/icon.ico](../../wx-ocr-src/src/icon.ico) is the source for the
resource and what `wxICON(orcgui)` resolves to in `MainFrame`'s
constructor.
