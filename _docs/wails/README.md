# Wails — Notes

Source: `repos-folder/wails`. Upstream: <https://github.com/wailsapp/wails>.

Wails is a Go framework for building desktop apps with a Go backend and a web-tech frontend (HTML/CSS/JS or React/Svelte/Vue/etc.), bundled into a single binary that renders in the OS-native WebView.

Two active lines: **v2** (stable) and **v3** (alpha — adds iOS / Android support, a new app/window/service API, and a Go-typed event system).
