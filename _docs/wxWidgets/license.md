# License

wxWidgets uses the **wxWindows Library Licence** — a modified version of
LGPL.

The key difference from stock LGPL: it explicitly permits distributing an
application that **statically links** wxWidgets without releasing the
application's source code. (Stock LGPL effectively requires either dynamic
linking or providing object files so users can relink.)

In practice this means:

- Free use in open-source projects ✔
- Free use in closed-source commercial projects, static or dynamic ✔
- You must keep wxWidgets's own license/copyright notices intact ✔
- Modifications to wxWidgets itself are still subject to LGPL-style
  share-back obligations ✔

Full text: `docs/licence.txt` in the upstream repo. Supporting files:
`docs/lgpl.txt`, `docs/gpl.txt`, `docs/preamble.txt`, `docs/licendoc.txt`.
