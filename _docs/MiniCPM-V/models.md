# Model Families

The repo documents two related model families.

MiniCPM-V focuses on efficient vision-language understanding. The current
root documentation highlights MiniCPM-V 4.6 as a 1.3B-parameter model for
single-image, multi-image, and video understanding. It emphasizes edge
deployment, mixed visual token compression, and small memory footprints.

MiniCPM-o extends the family into omnimodal interaction. The root docs
highlight MiniCPM-o 4.5 as a 9B-parameter model for vision, speech, and
full-duplex multimodal live streaming. The older MiniCPM-o 2.6 code is also
present in demos and evaluation scripts.

Older and legacy model pages are retained in `docs/`, including:

- MiniCPM-V 4.5
- MiniCPM-V 4.0
- MiniCPM-o 2.6
- MiniCPM-V 2.6
- MiniCPM-Llama3-V 2.5
- MiniCPM-V 2.0
- MiniCPM-V 1.0
- OmniLMM

The root model zoo lists model variants for Hugging Face and ModelScope,
including GPU, CPU GGUF, and quantized BNB/AWQ/GPTQ forms for newer models.
